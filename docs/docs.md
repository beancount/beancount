# Beancount Documentation Directory

## Overview
This directory is the designated location for the future text-based documentation of the Beancount project. Currently, the official documentation is hosted externally on Google Docs.

## Contents
*   **`README`**: Provides links to the current official documentation and the Google Docs source.

## Intention
The long-term goal for this directory is to contain a converted text version of the Google Docs documentation. This transition will allow the documentation to be:
1.  Version-controlled alongside the source code.
2.  Built and published to `beancount.github.io`.
