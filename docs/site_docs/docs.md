# Docs Directory

This directory is the designated location for Beancount's text-based documentation.

The current authoritative documentation sources still live externally in Google Docs:

- Documentation index: <http://furius.ca/beancount/doc/index>
- Google Docs source: <https://docs.google.com/document/d/1RaondTJCS_IUPBHFNdT8oqFKJjVJDsfsn6JEjBG04eA/>

## Purpose

The goal of this directory is to host documentation content alongside the Beancount source tree so it can be version-controlled and published from the main repository.
